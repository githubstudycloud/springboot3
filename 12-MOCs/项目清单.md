# 📋 项目清单

## 🔥 进行中的项目

### 工作项目
```dataview
TABLE 
  priority as "优先级",
  status as "状态",
  file.mtime as "最后更新",
  deadline as "截止日期"
FROM "02-Projects/工作项目"
WHERE status != "completed"
SORT priority DESC, deadline ASC
```

## ⏸️ 暂停的项目

```dataview
TABLE 
  priority as "优先级",
  file.mtime as "最后更新",
  reason as "暂停原因"
FROM "02-Projects"
WHERE status = "on-hold"
SORT file.mtime DESC
```

## ✅ 最近完成的项目

```dataview
TABLE 
  completed as "完成日期",
  duration as "历时",
  outcome as "成果"
FROM "02-Projects"
WHERE status = "completed"
SORT completed DESC
LIMIT 10
```
